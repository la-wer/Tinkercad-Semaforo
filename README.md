# Semaforo
> Um semáforo criado no tinkercad

✧ Sobre:

> - Ao botão ser clicado, o semáforo de pedestres abre por 5 segundos, com visor indicativo do timer
> - Depois de 5 ciclos de abertura do sinal, o buzzer toca um som
